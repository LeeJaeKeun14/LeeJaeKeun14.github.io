    검정과 유의확률

    • 검정(testing)은 데이터 뒤에 숨어있는 확률변수의 분포에 대한 가설이 맞는지 틀리는지 정량적으로 증명하는 작업이다.

    • 가설과 검정

      • 가설(hypothesis) : H
      • 검정(testing) : 가설을 증명하는 행위, 통계적 가설검정 이라고 한다.
      • 모수 검정(parameter testing) : 확률분포의 모숫값이 특정한 값을 가진다는 가설검정

    • 귀무가설

      • null hypothesis : 확률분포의 모수에 대한 가설($H_0$)
        • 데이터가 어떤 확률변수의 표본이라 가정한다
        • 데이터를 만드는 확률변수가 따르는 확률분포의 모수 $\theta$값이 어떤 특정한 실수값$\theta_0$로 고정되어 있다고 가정한다
        • 확률분포를 특정한 상태로 고정시켜야 하므로 반듯이 등식으로 표현해야 한다

    • 대립가설

      • alternative hypothesis : 우리가 주장하려는 혹은 귀무가설을 반박하려는 가설이 아닐 수 있다.($H_{\alpha}$)
        • 대립가설을 연구가설(research hypothesis)라고도 한다

    • 양측검정 유의확률

      • 검정의 관점에서 유의확률은
        귀무가설이 맞음에도 불구하고 현재 검정통계량값과 같은 혹은 대립가설을 더 옹호하는 검정통계량값이 나올 확률 이 식에서 $H_0$는 귀무가설이 진실인 사건을 뜻한다

    • 단측검정 유의확률

      • 모수 $\theta$에 대해서

        를 검정하고 싵다면 단측검정의 유의확률은 다음과 같이 구한다.

      • 우측검정(right-side test) : 모수 $\theta$가 특정한 값보다 크다는 것을 증명하는 경우
        유의확률 = $P(t\geq t_0|H_0) = 1 - F(t_0)$
        (이산확률분포일 경우) $1 - F(t_0 - 1)$

      • 좌측검정(left-side test) : 모수 $\theta$가 특정한 값보다 작다는 것을 증명하는 경우
        유의확률 = $P(t\leq t_0|H_0) = F(t_0)$
        (이산확률분포일 경우) $F(t_0 - 1)$

    유의수준과 기각역

    • level of significance : 계산된 유의확률값에 대해 귀무가설을 기각하는지 채택하는지를 결정할 수 있는 기준값(1%, 5%, 10% …)
      • 유의확률이 유의수준보다 작을경우 귀무가설을 기각하고 대립가설을 채택한다.

    이항검정

    • 이항검정은 이항분포를 이용하여 베르누이 확률변수의 모수 $\mu$에 대한 가설을 조사하는 검정 방법이다
      • scipy 코드
        scipy.stats.binom_test(x, n = None, p = 0.5, alternative = 'two-sided')
        x : 검정통계량, 1이 나온 횟수
        n : 총 시도 횟수
        p : 귀무가설의 $\mu$값
        alternative : 양측검정인 경우 ‘two-sided’, 단측검정인 경우 ‘one-sided’
    • 이항검정은 두 가지 값을 가지는 확률변수의 분포를 판단하는데 도움을 준다.

    카이제곱검정

    • 카테고리분포의 모수에 대해 검정을 하기 위해 사용(다항분포는 벡터를 반환)
      • 적합도 검정(goodness of fit test)이라고도 부른다.
      • scipy 코드
        scipy.stats.chisquare(f_obs, f_exe = None)
        f_obs : 데이터행렬
        f_exp : 기댓값 행렬, 디폴트일 경우 귀무가설은 $\mu = (\frac{1}{K},\dots,\frac{1}{K})$이다
