    엔트로피의 정의

    • entropy : 확률분포가 가지는 정보의 확신도 혹은 정보량을 수치로 표현한 것이다 이때 p(y) = 0인 경우 로그값이 정의되지 않음으로 극한을 이용한다
    • 엔트로피의 성질
      • 확률변수가 결정론적이면 확률분포에서 특정한 하나의 값이 나올 확률은 1, 이때 엔트로치는 0이 되고 최소값을 갖는다.
      • 최대값은 이산확률분포거 가질 수 있는 값이 $2^K$개면 엔트로피의 최대값은 각 값에 대한 확률이 모두 같은 값인 $\frac{1}{2^K}$이다. 따라서 엔트로피 값은
    • 가변길이 인코딩
      • 글자 “A, B, C, D” 인코딩
      • 글자 분포에 따른 가변길이 인코딩
    • 지니불순도
      • Gini impurity 엔트로피와 비슷하지만 로그연산이 없어 계산속도가 빨라 엔트로피 대신으로 자주 사용된다
    • 엔트로피 최대화
      • 조건 : 기댓값 0, 분산$\sigma^2$일때 엔트로피 를 가장 크게만드는 확률밀도함수는 정규분포 이다.
        확률밀도함수의 총면적 1 기댓값은 0 분산은 $\sigma^2$ 최적화 목적함수는 엔트로피이다
      • 라그랑주 승수법 풀이 변분법에서 도함수를 구하는 방법을 이용하고 연립방정식을 풀면

    상호정보량

    • mutual information

      • 두 확률변수 X, Y가 독립이면

        인 점과
        쿨백-라이블러 발산

        이 p, q가 같으면 0이되는 점을 이용해서 두 확률변수의 상관관계를 측정하는 방법이다.

      • 상호정보량

        상호 정보량은 엔트로피와 조건부엔트로피의 차이와 같다

    • 이산확률변수의 상호정보량
      • 상관관계가 있는 두 개의 카테고리 확률변수 X, Y에서 나온 표본 데이터가 N개 있다.
        • I : X의 카테고리 개수
        • J : Y의 카테고리 개수
        • $N_i$ : X = i인 데이터의 개수
        • $N_j$ : Y = j인 데이터의 개수
        • $N_{ij}$ : X = j, Y = j인 데이터의 개수이때 상호정보량은 다음과 같다
    • 최대정보 상관계수
      • 연속확률변수의 표본 데이터에서 상호정보량을 측정하려면 우선 확률분포함수를 알아야한다. 확률분포함수는 보통 히스토그램을 이용하는데 구간의 개수나 경계 위치에 따라 추정오차가 커징 수 있다. 이중 가장 큰 값을 선택하여 정규화한 것을 최대정보 상관계수(Maximal Information coefficient, MIC)라고 한다
