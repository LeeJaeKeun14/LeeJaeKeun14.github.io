    확률분포의 추론

    • 데이터 분석의 첫번째 가정.
      ‘분석하고자 하는 데이터가 어떤 확률변수로부터 실현된 표본이다’

    확률분포의 결정

    1. 확률변수가 어떤 확률분포를 따르는지 알아낸다

      • 데이터가 0, 1뿐이다
        $\to$ 베르누이분포
        데이터는 카테고리 값이어야 한다
        $\to$ 카테고리분포
        데이터는 0과 1사이의 실수 값이어야 한다
        $\to$ 베타분포
        데이터는 항상 0 또는 양수이어애 한다
        $\to$ 로그정규분포, 감마분포, F분포, 카이제곱분포, 지수분포, 하프코시분포 등
        데이터가 크기 제한이 없는 실수
        $\to$ 정규분포, 스튜던트t분포, 코시분포, 라플라스분포 등

    2. 데이터로부터 해당 확률분포의 모수의 값을 구한다

      • 모수추정 방법론 : 모수의 값으로 가장 가능성이 높은 하나의 숫자를 찾아내는 작업

    모수 추정 방법론

    • 모멘트 방법
    • 최대가능도 추정법
    • 베이즈 추정법

    모멘트 방법

    • method of moment : 표본자료에 대한 표본 모멘트가 확률분포의 이론적 모멘트와 같다고 생각하여 모수를 구한다
      • 기댓값위 식에서 N은 데이터의 개수, $x_i$는 표본 데이터다.
      • 분산
    • 베르누이분포
      • $\mu$
    • 카테고리분포
      • $\mu_k$
    • 정규분포
      • $\mu$
      • $\sigma^2$
    • 베타분포
      • $\mu$
      • $\sigma^2$이 연립방정식을 풀면
    • 다변수정규분포
      • $\mu$
      • $\Lambda^{-1}$

    최대가능도 추정법

    • 가능도함수(likelihood function)
      확률분포 X에 대한 확률밀도함수(확률질량함수)를 다음으로 정의한다

      이때 x를 알고있는 상수계수로 놓고 $\theta$를 변수로 계산하는 함수를 가능도 함수라고 한다

      • 확률밀도함수에서 모수를 변수로 보는 경우에 이 함수를 가능도함수(likelihood function)라고 한다.

      • 확률밀도함수는 적분하면 전체 면적이 1이 되지만, 가능도함수는 가능한 모든 모숫값 $\theta$에 대해 적분했을 떄 1이 된다는 보장이 없다

      • 가능도함수와 확률밀도 함수

        • 확률밀도함수 f(x;$\theta$)
          • $\theta$ 값을 이미 알고 있음
          • $\theta$는 상수, x는 변수
          • $\theta$가 이미 정해져 있는 상황에서 x값의 상대적 확률
          • 적분하면 전체 면적은 항상 1
        • 가능도함수 L($\theta$) = p(x|$\theta$)
          • x가 이미 발생. 값을 이미 알고 있음
          • x는 상수, $\theta$는 변수
          • x가 이미 정해져 있는 상황에서 $\theta$값의 상대적 확률
          • 적분하면 전체 면적이 1이 아닐 수 있다.

    • 카테고리분포의 최대가능도 모수 추정

    • 최대 가능도 추정법

      • Maximum Likelihood Estimation(MLE)
        주어진 표본에 대해 가능도를 가장 크게하는 모스 $\theta$를 찾는 방법

    • 복수의 표본 데이터가 있는 경우의 가능도함수

    • 로그가능도함수

      • 일반적으로 최대가능도 추정법을 사용하여 가능도가 최대가 되는 $\theta$를 계산하려면 수치적 최적화를 해야한다
      • 이때 곱으로 나타나는경우를 로그변환을 이용하여 덧셈계산으로 단순해 진다(LL = log L)

    • 베르누이 분포의 최대가능도 모수 추정

      • 모수가 $\mu$인 베르누이 분포의 확률질량함수는 다음과 같다

      • N번 반복 시행으로 표본 데이터가 $x_1, \dots, x_N$이 있는 경우에는 모두 독립이므로 전체 확률질량함수는 각각의 확률질량함수의 곱과 같다.

      • 미분을 쉽게 하기 위해 로그 변환을 하여 로그가능도를 구하면 다음과 같다.

      • x = 1(성공) 또는 x = 0(실패) 이므로 횟수를 다음과 같이 표기한다.

        로그가능도는 다음과 같아진다.

        이 목적함수를 모수로 미분한 값이 0이 되게 하는 모수값을 구하면 다음과 같다

        최대가능도 추정법에 의한 베르누이분포의 모수는 1이 나온 횟수와 전체 시행횟수의 비율이다.

    베이즈 추정법

    • Bayesian estimation
      모수값이 가질 수 있는 모든 가능성의 분포를 계산하는 작업

    • 베이즈 추정법의 기본 원리

      • p($\mu$) : 모수의 사전분포
        베이지안 추정 작업전 이미 알고있는 $\mu$의 분포를 뜻한다. 모수나 분포에 대해 지식이 없을경우 균일분포 Beta(1,1)이나 무정보분포인 정규분포N(0,$\sigma^2$)을 사용한다
      • p($\mu|x_1,\dots,x_N$) : 모수의 사후분포
        데이터가 주어진 상태에서의 $\mu$에 대한 조건부 확률 분포 이다
      • p($x_1,\dots,x_N|\mu$) : 가능도(likelihood)분포
        모수 $\mu$가 특정한 값으로 주어졌을 때 주어진 데이터가 나올 수 있는 확률값을 나타낸다
      1. 모수적(parametric) 방법
        모수 분포를 표현하는 확률분포함수의 모수를 하이퍼모수라 하는데 이를 계산하는 작업
      2. 비모수적(non parametric) 방법
        MCMC(Markov chain Monte Carlo)와 같은 몬테카를로 방법이 있다
    • 베르누이분포의 모수 추정
      • 데이터는 모두 독립적인 베르누이 분포의 곱이므로 가능도 함수는 다음과 같다
      • 베이즈 정리를 사용하면 사후분포가 다음처럼 갱신된 하이퍼모수 $a’, b’$를 가지는 또다른 베타분포가 된다.이러한 사전분포를 켤레 사전확률분포(conjugate prior)라고 한다
      • 갱신된 하이퍼 모수값
      • 또한 베이즈 추정법의 장점은 순차적(sequential)계산이 가능하다는 점이다
    • 카테고리분포의 모수 추정
      • 클래스 개수가 K인 카테고리분포의 모수 $\mu$벡터를 베이즈 추정법으로 추정해보자.데이터는 모두 독립인 카테고리 분포이므로베이즈 정리로 사후 분포를 구하면 다음과 같이 갱신된 하이퍼모수 $\alpha’_i$를 가지는 디리클리분포가 된다결과가 디리클리분포가 켤레분포임을 알 수 있다
      • 갱신된 하이퍼 모수값
    • 정규분포의 기댓값 모수 추정
