    최적화 문제

    • 최적화 문제는 함수의 $f$의 값을 최대화 혹은 최소화하는 변수 x의 값 $x^*$를 찾는 것이다.

      이 값 $x^*$를 최적화 문제의 해(solution)이라고 한다.

      • $f(X)$를
        • J : 목적함수(objective function)
        • C : 비용함수(cost function)
        • L : 손실함수(loss function)
        • E : 오차함수(error function)

    그리드 서치

    • Grid search

      목적함수의 값을 가장 작게 하는 x 위치를 찾는 최적화 문제.
      가능한 모든 x의 값을 넣어 보고 그중 가장 작은(큰) 값을 선택한다.

    수치적 최적화

    • numerical optimization

      반복적 시행 착오(trial and error)에 의해 최적화 필요조건을 만족하는 $x^*$를 찾는 방법.

      두 가지 알고리즘이 필요하다

      • 현제 위치 $x_k$가 최적점인지 판단하는 알고리즘
      • 어떤 위치 $xk$를 시도한 뒤, 다음 번에 시돠할 위치 $x{k+1}$을 찾는 알고리즘

    • 기울기 필요조건

      현재 위치가 최소점인지 아닌지 판단하는 알고리즘.

      어떤 독립 변수 값 $x^$가 최소점이려면 일단 값 $x^$에서 함수의 기울기와 도함수$\frac{df}{dx}$값이 0이라는 조건을 만족해야 한다. 이를 기울기 필요조건 이라고 한다.

      다변수 함수인 경우 모든 변수에 대한 편미분값이 0이어야 한다.

      즉

      이때 그레디언트 벡터 $\nabla f$를 g라는 기호로 간단하게 표시한다

    • 최대경사법 문제

      • steepest Gradient Descent

        현재 위치 $xk$에서의 기울기 값 $g(x_k)$만을 이용하여 다음번 위치 $x{k+1}$를 결정하는 방법이다

        만약 현재 위치가 $x_k$에서 기울기가 음수면 $g(x_k) < 0$이므로 앞으로 진행하고 $x_k$에서 기울기가 양수면 $g(x_k) > 0$이므로 뒤로 진행하게 되어 점점 낮은 위치로 옮겨간다.
        이때 $\mu$를 스텝 사이즈 라고 한다.

    • 2차 도함수를 사용한 뉴턴 방법
      • Newton method
        목적함수가 2차 함수라는 가정하에 한 번에 최저점을 찾는 방법이다. 그레디언트 벡터에 헤시안 행렬의 역행렬을 곱해서 방향과 거리가 변형된 그레디언트 벡터를 사용한다.

    • 수치적 최적화 문제

      • 진동(oscillation)현상
        곡면의 모양이 계곡(valley)와 같이 생긴 경우, 즉 그레디언트 벡터가 최저점을 가리키고 있지 않은 경우 특정 구간에서 이동을 반복하는 현상.

      • 전역 최적화 문제
        만약 함수가 복수의 국소 최저점을 가지고 있는 경우 수치적 최적화 방법으로 항상 전역 최저점에 도달 할 수 없다.

      • 컨벡스 문제
        목적함수의 2차 도함수의 값이 항상 0 이상이 되는 영역에서만 정의된 최적화 문제를 컨벡스문제 라고 한다

        다변수 목점함수의 경우에는 주어진 영역에서 헤시안 행렬이 항상 양의 준정부호이라는 조건이 된다.

        컨벡스 문제에서는 항상 전역 최저점이 존재한다.
